# Muffix Sassif - Team Reference Document

## Code rules
* _**Do not use dollars**_. Use `'#'` or `$\dollar$` instead.
* You can use Latex inside code (comments). `$a_1, ..., a_n$` will be displayed as $a_1, ..., a_n$.


## Generate pdf
To generate new `.pdf` and `.tex` files run
```sh
sh gen.sh
```
